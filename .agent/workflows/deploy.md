# Deploy to GitHub Pages

// turbo-all

1. Stage all changes
```bash
cd ~/日常小程式/宇羲英文 && git add -A
```

2. Commit with a descriptive message
```bash
cd ~/日常小程式/宇羲英文 && git commit -m "update sight word app"
```

3. Push to GitHub
```bash
cd ~/日常小程式/宇羲英文 && git push origin main
```

After pushing, GitHub Pages will automatically update in about 1 minute.
The site will be available at: https://<your-username>.github.io/yuxi-english/
